Pigeon (W.I.P.)
pigeon is an email that helps organizations (like campuses and companies) send targeted, interest-based emails to members. instead of using listservs or other static mailing lists, pigeon allows users to send any email to only the people who might be interested in the content of the email. this is done through manual and automatic content tagging. it keeps your inbox free of cluttered advertising messages you don't care about and allows you to target your messages to the specific people who will care.

we are using Node.js and a PostgreSQL database. we are also building a chrome extension that uses an open source gmail js library, which, when enabled, will allow users to access pigeon's functionality from their own gmail inbox. my focus is on the backend/database and the chrome extension.

this project is a work in progress. check out the proposal. check back later to see the app.
