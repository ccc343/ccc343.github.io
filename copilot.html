CoPilot
copilot is a chrome extension that enables groups to browse collaboratively on the web. co-browsing is a new paradigm for group collaboration and productivity - users can create their own shared browsing sessions with others and collaboratively browse the web. imagine being able to plan a trip or research a group project without having to send lists of links to each out. shared browsing sessions occur in a separate chrome window, in which all tabs are mirrored between all participants. actions are mirrored as well.

we built this using a firebase backend to meet the need for a real-time, synchronized database that will keep all clients' tabs matched up. my focus here was writing javascript for the chrome extension. we will be revamping this 24hr-hackathon project soon.
